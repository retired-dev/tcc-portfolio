# Portfólio para TCC em Desenvolvimento Full Stack

Este repositório organiza os estudos, exercícios e exemplos produzidos ao longo da formação em Desenvolvimento Full Stack. Cada pasta corresponde a uma disciplina e contém os códigos discutidos no Trabalho de Conclusão de Curso.

O repositório está organizado em torno das cinco disciplinas que compõem o **Capítulo 2 — Organização do Portfólio** do TCC.

---

## Estrutura

```
portfolio-tcc/
├── 01-poo/                  → Programação Orientada a Objetos
├── 02-server-side/          → Arquitetura Server-Side
├── 03-web/                  → Programação para Web
├── 04-web-services/         → Web Services
└── 05-banco-dados/          → Banco de Dados Relacional
```
